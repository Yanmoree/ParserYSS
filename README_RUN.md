## Первый запуск

1. Запустите приложение одним из способов выше
2. Откройте Telegram и найдите бота: `@yss_parser_bot`
3. Отправьте команду `/start`
4. Бот зарегистрирует вас как нового пользователя

## Важно!

⚠️ **По умолчанию новые пользователи создаются БЕЗ активной подписки**

Для активации подписки нужно:
1. Первый пользователь должен быть назначен администратором вручную
2. Администратор может управлять подписками других пользователей через админ-панель

### Как сделать первого пользователя админом:

1. Запустите бота и зарегистрируйтесь через `/start`
2. Остановите бота (Ctrl+C)
3. Откройте файл `data/users.json`
4. Найдите своего пользователя и измените:
   ```json
   "isAdmin": true,
   "hasLifetimeSubscription": true
   ```
5. Сохраните файл и запустите бота снова

## Основные команды бота

- `/start` - Главное меню
- `/settings` - Настройки парсера
- `/queries` - Управление поисковыми запросами
- `/admin` - Админ-панель (только для администраторов)

## Troubleshooting

### Ошибка "Bot token is invalid"
- Проверьте правильность токена в `TelegramBotService.java`
- Убедитесь, что токен не содержит лишних пробелов

### Ошибка компиляции
- Убедитесь, что установлена Java 17 или выше: `java -version`
- Убедитесь, что установлен Maven: `mvn -version`

### Парсер не находит товары
- Проверьте, что у пользователя активна подписка
- Проверьте, что добавлены поисковые запросы
- Проверьте, что выбраны платформы в настройках
- Проверьте логи на наличие ошибок

### Бот не отвечает
- Проверьте, что приложение запущено
- Проверьте логи на наличие ошибок подключения
- Убедитесь, что есть интернет-соединение
