# ParserYSS - Многопользовательский парсер товаров

## Описание

ParserYSS - это многопользовательский Telegram бот для автоматического парсинга товаров с различных платформ:
- **Avito** (Россия)
- **Mercari** (Япония)
- **Goofish** (Китай)

## Возможности

### Для пользователей:
- ✅ Автоматический парсинг новых товаров
- ✅ Настройка фильтров (цена, возраст товара)
- ✅ Выбор платформ для парсинга
- ✅ Управление поисковыми запросами
- ✅ Получение уведомлений о новых товарах в реальном времени

### Для администраторов:
- ✅ Управление пользователями
- ✅ Настройка доступа к платформам
- ✅ Управление подписками
- ✅ Статистика использования

## Архитектура

### Компоненты:

1. **Model Layer** - модели данных
   - User - пользователь
   - UserSettings - настройки парсера
   - Query - поисковый запрос
   - Product - товар
   - ParserState - состояние парсера

2. **Storage Layer** - хранение данных
   - FileStorage - файловое хранилище
   - UserRepository - управление пользователями
   - QueryRepository - управление запросами
   - ParserStateRepository - хранение состояний
   - UserSettingsRepository - управление настройками

3. **Service Layer** - бизнес-логика
   - AvitoHeaderService - получение cookies для Avito
   - MercariHeaderService - получение headers для Mercari
   - TwoCaptchaService - автоматическое решение капчи
   - ParserManager - управление парсингом

4. **Parser Layer** - парсеры
   - AvitoApiParser - парсер Avito
   - MercariParser - парсер Mercari
   - GoofishParser - парсер Goofish

5. **Bot Layer** - Telegram бот
   - TelegramBotService - основной сервис бота
   - Handlers - обработчики команд и callback
   - Keyboards - клавиатуры для UI

## Установка

### Требования:
- Java 17+
- Maven 3.6+
- Chrome/Chromium (для Selenium)

### Настройка:

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd ParserYSS
```

2. Установите зависимости:
```bash
mvn clean install
```

3. Настройте переменные окружения:
```bash
export TWOCAPTCHA_API_KEY="your_2captcha_api_key"
```

4. Запустите приложение:
```bash
mvn exec:java -Dexec.mainClass="com.parseryss.Main"
```

## Использование

### Для пользователей:

1. Найдите бота в Telegram: `@ParserYSSBot`
2. Отправьте `/start` для регистрации
3. Дождитесь активации подписки от администратора
4. Настройте платформы и фильтры
5. Добавьте поисковые запросы
6. Запустите парсер кнопкой "▶️ Запустить парсер"

### Для администраторов:

1. Откройте админ панель: "👑 Админ панель"
2. Управляйте пользователями
3. Настраивайте доступ к платформам
4. Управляйте подписками

## Логика работы

### Парсинг новых товаров:

1. При первом запуске:
   - Находится самый новый товар
   - Сохраняется его время публикации и ID
   - Возвращается 1 товар как точка отсчета

2. При следующих запусках:
   - Парсятся товары новее сохраненного времени
   - Если время совпадает, проверяется ID
   - Обновляется время последнего товара

3. При остановке:
   - Состояние сохраняется автоматически
   - При следующем запуске парсинг продолжается с того же места

## Технические детали

### Получение cookies:

- **Avito**: Selenium перехватывает cookies через Chrome
- **Mercari**: Selenium перехватывает headers через Chrome Performance Logs
- **Goofish**: Генерация подписей для API запросов

### Решение капчи:

- Автоматическое решение через 2Captcha API
- Поддержка reCAPTCHA v2 и hCaptcha
- Стоимость: ~$1 за 1000 решений

### Многопользовательский режим:

- ExecutorService для параллельного выполнения
- Каждый пользователь парсится в отдельном потоке
- Индивидуальные настройки и интервалы

## Разработка

### Добавление нового парсера:

1. Создайте класс, реализующий `SiteParser`
2. Реализуйте методы `search()` и `searchNewProducts()`
3. Добавьте парсер в `ParserFactory`

### Добавление новой команды бота:

1. Создайте handler в `com.parseryss.bot.handlers`
2. Добавьте обработку в `TelegramBotService`
3. Создайте клавиатуру в `com.parseryss.bot.keyboards`
