# 🚀 Быстрый старт ParserYSS

## Шаг 1: Запустите бота

```bash
cd /Users/yanmore/IdeaProjects/ParserYSS
chmod +x build_and_run.sh
./build_and_run.sh
```

Или если уже скомпилировано:

```bash
chmod +x run.sh
./run.sh
```

## Шаг 2: Откройте Telegram

1. Найдите бота: **@yss_parser_bot**
2. Нажмите "Start" или отправьте `/start`

## Шаг 3: Станьте админом

⚠️ **ВАЖНО**: По умолчанию новые пользователи создаются без подписки!

### Чтобы сделать себя админом:

1. **Остановите бота** (нажмите `Ctrl+C` в терминале)

2. **Откройте файл**:
   ```bash
   nano data/users.json
   ```
   Или в любом текстовом редакторе

3. **Найдите своего пользователя** и измените:
   ```json
   {
     "userId": 123456789,
     "username": "user_123456789",
     "isAdmin": true,  // ← Измените на true
     "subscriptionExpiry": "",
     "hasLifetimeSubscription": true,  // ← Измените на true
     "allowedPlatforms": [],
     "createdAt": "...",
     "lastActive": "..."
   }
   ```

4. **Сохраните файл** и **запустите бота снова**:
   ```bash
   ./run.sh
   ```

## Шаг 4: Настройте парсер

В Telegram боте:

1. **Настройки** (`/settings`):
   - Выберите платформы (Avito, Mercari, Goofish)
   - Установите фильтры по цене
   - Настройте интервал проверки

2. **Запросы** (`/queries`):
   - Добавьте поисковые запросы
   - Например: "iPhone 15", "MacBook Pro", "Ноутбук"

3. **Запустите парсер**:
   - Нажмите кнопку "🚀 Запустить парсер"

## Шаг 5: Получайте товары!

Парсер автоматически будет отправлять вам новые товары в Telegram!

## 📊 Дополнительные команды

- `/start` - Главное меню
- `/settings` - Настройки
- `/queries` - Запросы
- `/admin` - Админ-панель (только для админов)

## 👥 Управление пользователями (для админов)

Через `/admin` вы можете:
- Просматривать всех пользователей
- Активировать/деактивировать подписки
- Выбирать доступные платформы для каждого пользователя
- Устанавливать срок действия подписки

- `logs/parseryss.log` (файл)
